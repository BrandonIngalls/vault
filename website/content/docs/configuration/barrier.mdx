# `barrier` Stanza

The `barrier` stanza specifies optional configurations of the cryptographic 
barrier.  

```hcl
barrier {
  key_rotation_interval = "2160h"
}
```

## `barrier` Parameters

- `key_rotation_max_operations` `(int: "3865470566")` - Specifies the maximum
  number of encryptions performed by the barrier key prior to automatic rotation.
  The default is 10% fewer than recommended by NIST 800-38d.  Only smaller values
  than the default are possible.
- `key_rotation_interval` `(string: "")` - The maximum time a barrier key term
  can operate before being automatically rotated.  By default the barrier key
  is not rotated on the basis of time.
